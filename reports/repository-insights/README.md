## 총평

기존 표가 “어디에 무엇이 있는지”를 보여줬다면, 이 보고서는 “각 레포를 어떤 관점으로 읽어야 하는지”를 정리합니다. 각 row의 인사이트와 위험 신호를 먼저 보고, 보고서/소스 링크로 들어가면 조사 시간을 줄일 수 있습니다. CSV와 JSON은 전체 1,136개 레포를 빠짐없이 담습니다.
