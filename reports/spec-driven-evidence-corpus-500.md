## Method Limitations

- OpenAlex search ranking is broad. The corpus includes adjacent fields such as requirements engineering, BDD/TDD, program synthesis, MBSE, API contracts, traceability, and agent memory because SDD reuses these older research lines.
- The 500+ count is an evidence inventory, not an assertion that every work directly advocates AI-era SDD.
- For final decisions, use the curated/key-source section in the comparison report and inspect the JSON metadata for each candidate.
