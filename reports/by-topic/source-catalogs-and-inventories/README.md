# 소스 카탈로그/인벤토리

생성 시각: 2026-06-18T15:47:35.662Z

조사에 사용한 소스 카탈로그, 인벤토리, 클론 목록, evidence ledger입니다.

## 요약

- 조사 단위: `source-catalogs-and-inventories` 주제에 속한 보고서 묶음입니다.
- 포함 범위: 6개 보고서이며, 주요 보고서 유형은 overview-report (4), presentation-report (2)입니다.
- 주요 출처 폴더: root (4), presentations (2).

## 총평

이 README는 같은 주제에 흩어진 보고서를 한 번에 따라가도록 만든 주제형 입구입니다. 추천 읽기 경로는 먼저 읽을 보고서를 우선순위로 보여주고, 전체 보고서는 빠짐없는 목록을 제공합니다. 같은 보고서가 여러 주제에 걸칠 수 있으므로, 큰 흐름은 이 페이지에서 보고 세부 파일 위치는 폴더 값을 확인하는 방식이 좋습니다.
