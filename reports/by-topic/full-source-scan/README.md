# 전체 소스 스캔/코드 검색

생성 시각: 2026-06-18T15:38:53.459Z

코딩 에이전트가 소스를 발견하고 인덱싱하고 사용자 의도와 코드 위치를 연결하는 방식입니다.

## 요약

- 조사 단위: `full-source-scan` 주제에 속한 보고서 묶음입니다.
- 포함 범위: 3개 보고서이며, 주요 보고서 유형은 overview-report (2), source-scan-report (1)입니다.
- 주요 출처 폴더: root (2), full-source-scan (1).

## 총평

이 README는 같은 주제에 흩어진 보고서를 한 번에 따라가도록 만든 주제형 입구입니다. 추천 읽기 경로는 먼저 읽을 보고서를 우선순위로 보여주고, 전체 보고서는 빠짐없는 목록을 제공합니다. 같은 보고서가 여러 주제에 걸칠 수 있으므로, 큰 흐름은 이 페이지에서 보고 세부 파일 위치는 폴더 값을 확인하는 방식이 좋습니다.
